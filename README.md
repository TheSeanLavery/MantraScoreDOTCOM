# MantraScore.com

[![MIT License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Website](https://img.shields.io/badge/Website-mantrascore.com-green.svg)](https://mantrascore.com)

MantraScore is a web application that helps you track your daily affirmations and mindful speech patterns. By becoming more conscious of what you say, you can cultivate more positive thinking and self-talk.

## Features

- **Affirmation Tracking**: Set positive affirmations with targets and track how often you say them
- **Speech Pattern Awareness**: Monitor words or phrases you want to avoid
- **Daily Progress**: Calendar view to see your progress day by day
- **Data Management**: Export and import your data for backup and portability
- **Privacy-Focused**: 100% private - all data is stored locally in your browser

## Technology Stack

- React + TypeScript
- IndexedDB for client-side storage
- Web Speech API for voice recognition
- TailwindCSS for styling

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/TheSeanLavery/mantrascore.com.git
cd mantrascore.com

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Privacy

MantraScore is designed with privacy in mind. All data is stored locally in your browser using IndexedDB. No audio recordings or transcripts are sent to any server.
